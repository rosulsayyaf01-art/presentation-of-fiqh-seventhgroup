Website ini dirancang sebagai Single File Application, artinya semua kode (HTML, CSS, JS) berada dalam satu file index.html. Ini memudahkan proses deployment.
Langkah 1: Persiapan
 * Buat folder baru di komputer Anda, misal ruang-baca.
 * Simpan kode website sebagai index.html di dalam folder tersebut.
Langkah 2: Upload ke GitHub
 * Buat repository baru di GitHub (misal: ruang-baca).
 * Upload file index.html ke repository tersebut.
 * (Opsional) Tambahkan file README.md ini agar repository terlihat profesional.
Langkah 3: Aktifkan GitHub Pages
 * Buka tab Settings di repository Anda.
 * Pilih menu Pages di sidebar kiri.
 * Pada bagian Source, pilih branch main (atau master).
 * Klik Save.
 * Tunggu beberapa saat, link website Anda akan muncul (biasanya https://username.github.io/ruang-baca).
